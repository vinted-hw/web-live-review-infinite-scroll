# WEB-4821 — Infinite scroll for photo gallery

**Type:** Story  
**Priority:** Medium  
**Reporter:** Product  
**Assignee:** —  
**Sprint:** Sprint 34  

---

## What we want

Instead of just showing one page of photos, the gallery should keep loading more as the user scrolls down. Like Instagram basically.

## Acceptance Criteria

- [ ] More photos load when user scrolls down
- [ ] Show some kind of loading thing while fetching
- [ ] Should work on mobile too

## Notes

The API has pagination so use that. Pexels returns a `next_page` field I think.

## Design

No designs, just make it look consistent.
